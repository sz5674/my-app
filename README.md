# English Study App 取扱説明書

## 1. アプリ概要

このアプリは、TOEIC学習の基礎力を強化するためのブラウザ学習アプリです。  
以下の学習モードを、タブで切り替えて利用できます。

- 英単語テスト（4択）
- 短文穴埋め問題（Part 5風）
- 長文穴埋め問題（Part 6: 本文補完＋文挿入）
- 長文読解問題（Part 7）
- ミニ模試（10問タイマー）

加えて、`記録` タブで日別進捗・正答率・弱点を確認できます。

本アプリは TOEIC Reading の公開形式（Part 5 / Part 6 / Part 7）を意識した単元学習に対応しています。

## 3. 画面構成

- `ホーム`  
学習メニューの概要を表示
- `記録`  
学習ダッシュボード（累計、今日の進捗、連続学習日数など）
- `英単語テスト`  
単語と意味の4択
- `短文穴埋め問題`  
文法・語法中心の4択
- `長文穴埋め問題`  
本文補完（Part 6）を解く形式
- `長文読解問題`  
英文パッセージを読んで設問に回答
- `ミニ模試`  
制限時間内に10問解く模試形式（タブの最後に配置）

---

## 4. 基本的な使い方

1. 任意のタブを開く
2. 問題文を読み、選択肢を1つ選ぶ
3. 色分け（青=正解、赤=誤答）とラベルで正誤を確認
4. 必要に応じて `次の問題` で進む

`短文穴埋め問題` / `長文穴埋め問題` / `長文読解問題` では、回答時に解説が表示されます。  
`英単語テスト` では解説を表示しません。

必要なタブでは `TOEIC単元` セレクトで、解きたい単元だけに絞って学習できます。

## 5. 各モードの詳細

### 5.1 英単語テスト

- 英単語に対して、日本語の意味を選ぶ形式
- 学習履歴（正答数/回答数）は自動保存

### 5.2 短文穴埋め問題（Part 5風）

- 品詞・前置詞・接続詞・時制などを学習
- 回答時に解説を表示

### 5.3 ミニ模試（10問）

- 10問連続の実戦形式
- 出題内訳（縮小版）: Part 5×2 / Part 6×3 / Part 7シングル×2 / Part 7ダブル×1 / 語彙×2
- 全体タイマーあり（4分/6分/8分）
- 終了時に以下を表示:
  - スコア
  - 正答率
  - カテゴリ別結果
  - 全問題の答え合わせ（自分の回答/正解/解説）
- 誤答問題の解き直しに対応

### 5.4 長文穴埋め問題（Part 6）

- TOEIC Part 6を意識した「本文補完」形式
- 1つの本文内で4問構成（語句補完×2、文挿入×1、語形×1）
- 回答時に解説を表示

### 5.5 長文読解問題（Part 7）

- 短文パッセージを読み、内容一致問題に回答
- 回答時に解説を表示
- シングル/ダブルパッセージ形式を含む

### 5.6 問題数バランス（現在）

- 英単語テスト: 500問
- 短文穴埋め問題（Part 5）: 約120問
- 長文穴埋め問題（Part 6）: 64問
- 長文読解問題（Part 7）: 約120問

---

## 6. 記録（学習ダッシュボード）

`記録` タブでは、次の情報を確認できます。

- 今日の進捗（目標に対する達成率）
- 総回答数・総正答率
- セクション別正答率（穴埋め/長文）
- 直近の弱点カテゴリ・誤答しやすい選択肢
- 連続学習日数、累計学習日数
- 直近7日の日別記録

`目標を保存` で、1日の目標問題数を変更できます。

---

## 7. データ保存について

- 学習データはブラウザの `localStorage` に保存されます
- ブラウザデータを削除すると、学習記録も消えます
- 同じブラウザ・同じ端末なら、次回起動時も続きから再開できます

---

## 8. よくある使い方（おすすめ）

1. `英単語テスト` で語彙をウォームアップ
2. `短文穴埋め問題` で文法・語法を強化
3. `ミニ模試` で時間感覚を確認
4. `記録` タブで弱点を確認
5. `長文穴埋め問題` / `長文読解問題` で復習

---

## 9. トラブルシューティング

- **記録が更新されない**
  - `記録` タブの `記録を更新` ボタンを押してください
- **以前のデータが消えた**
  - ブラウザのキャッシュ/サイトデータ削除の影響の可能性があります
- **表示が崩れる**
  - ブラウザを再読み込みし、`index.html` と `styles.css` が同じフォルダにあるか確認してください
