# English Study App 取扱説明書

## 1. 概要

ブラウザだけで使う TOEIC **Reading（Part 5・6・7）** 相当の練習と語彙です。

**Part 1〜4** は本試験では **Listening** です。本アプリにはリスニング・スピーキング・ライティングは含みません。

| タブ | 内容 |
|------|------|
| `英単語` | 英単語と意味の4択 |
| `短文穴埋め問題` | Part 5（文法・語法） |
| `長文穴埋め問題` | Part 6（本文補完・文挿入） |
| `長文読解問題` | Part 7（シングル／ダブル） |
| `ミニ模試` | 10問・タイマー（上記のミックス） |
| `記録` | 進捗・正答率・弱点の確認 |

各タブの操作は同じです。ホームから各タブへ飛べます。

## 2. 起動

- GitHub Pages: [https://sz5674.github.io/my-app/](https://sz5674.github.io/my-app/)

---

## 3. 操作の流れ

1. タブを選ぶ（ホームのショートカットからでも可）
2. 問題を読み、選択肢を1つ選ぶ
3. 正誤は色（青＝正解、赤＝不正解）とバッジで表示
4. `次の問題` で続ける

短文／長文穴埋め／長文読解では回答後に**解説**が出ます。`英単語` では解説は出しません。  
単元の絞り込みは、該当タブの **TOEIC単元** から行います。

---

## 4. ミニ模試（10問）

出題の内訳（縮小版）: Part 5×2 / Part 6×3 / Part 7シングル×2 / Part 7ダブル×1 / 語彙×2  
タイマーは 4分 / 6分 / 8分から選択。終了後にスコア・カテゴリ別結果・全問の答え合わせが表示され、誤答の解き直しに対応します。

---

## 5. 問題数

件数はデータ更新で変わります。正確な内訳はアプリのホーム「問題数（現在）」を参照してください。

---

## 6. 記録タブ

今日の目標（1日の目標問題数）、総正答率、穴埋め・長文・Part 6 の正答率、弱点カテゴリ、連続学習日数などを表示します。`目標を保存` で1日の目標を変更できます。

---

## 7. データ

`localStorage` に保存されます。ブラウザのサイトデータを消すと記録も消えます。

---

## 8. トラブルシューティング

- **記録が古い** → `記録を更新` を押す
- **表示がおかしい** → 再読み込み。`index.html`・`styles.css`・`app.js` が同じフォルダか確認

---

## 9. 拡張について

リスニングなどは未実装ですが、タブ構成はあとからモードを足しやすい形にしています。
